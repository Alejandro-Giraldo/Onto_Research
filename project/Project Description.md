

# Ontology Research Lab



Este repositorio apunta a ser un proceso en el que se cuenta con:

1. Proceso de Ingestión de documentos no estructurados a grafo.
2. Proceso de consulta de grafo existente en (cypher, RDF) y enriquecimiento de contexto.
3. Uso de contexto para la creación de agente.

La forma de probar es crear un agente sin contexto sobre algo, y crear otro agente con contexto.
